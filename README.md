# 🏥 Healthcare System Stress Intelligence

Healthcare System Stress Intelligence is an end-to-end Machine Learning project developed to analyze healthcare-related indicators and estimate the stress experienced by healthcare systems across different countries. The project combines data preprocessing, feature engineering, machine learning models, visualization, and automated report generation to transform raw healthcare data into meaningful insights that support healthcare planning and policy decisions.

The project follows a modular architecture where every stage of the Machine Learning pipeline is implemented as an independent Python module. This improves readability, maintainability, and scalability while demonstrating a complete real-world data science workflow from data collection to prediction and reporting.

## **data/**

The **data** directory stores every dataset used throughout the project. It separates the original dataset from processed datasets to maintain a clean and reproducible workflow.

### **raw/**

The **raw** folder contains the original healthcare dataset exactly as it was collected without any modifications.

#### **Global_Mental_Health_Crisis_Index_2026.csv**

This is the primary dataset used throughout the project. It contains healthcare indicators from different countries and acts as the input for preprocessing, feature engineering, model training, and prediction.

---

### **processed/**

The **processed** folder stores datasets generated after cleaning and transforming the original data.

#### **cleaned_mental_health.csv**

This dataset is produced after preprocessing by removing duplicate records, handling missing values, and preparing the data for Machine Learning.

#### **clustered_dataset.csv**

This dataset contains all processed records along with cluster labels assigned by the K-Means clustering algorithm.

#### **final_dataset.csv**

This is the final feature-engineered dataset used for training regression and classification models.
