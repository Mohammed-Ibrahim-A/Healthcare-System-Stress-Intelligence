# 🏥 Healthcare System Stress Intelligence

Healthcare System Stress Intelligence is a Machine Learning project that analyzes global mental health and healthcare data to measure healthcare system stress. The project performs data preprocessing, feature engineering, visualization, regression, classification, clustering, and business analysis to generate meaningful healthcare insights.

The system predicts Healthcare Stress Scores, classifies stress levels into categories, discovers hidden regional patterns using clustering, identifies important influencing factors, and generates reports to support healthcare policy decisions.

# 📁 data

The **data** folder stores every dataset used throughout the project. It separates the original dataset from the processed datasets to maintain a proper Machine Learning workflow.
---
## 📁 raw

The **raw** folder contains the original dataset before any preprocessing or feature engineering.

### 📄 Global_Mental_Health_Crisis_Index_2026.csv

This is the original healthcare dataset used as the main input for the entire project. Every preprocessing, prediction, visualization, and report is generated from this dataset.
---
## 📁 processed

The **processed** folder stores datasets generated after cleaning, transformation, and Machine Learning processing.

### 📄 cleaned_mental_health.csv

Contains the cleaned version of the original dataset after removing missing values, correcting data types, and preparing the data for analysis.

### 📄 clustered_dataset.csv

Stores the dataset after applying clustering algorithms. Each record contains its assigned healthcare region cluster.

### 📄 final_dataset.csv

Contains the final engineered dataset that is used for training Machine Learning models and generating predictions.

# ▶️ Installation

Clone the repository.

```bash
git clone https://github.com/Mohammed-Ibrahim-A/Healthcare-System-Stress-Intelligence.git
```

Move into the project directory.

```bash
cd Healthcare-System-Stress-Intelligence
```

Install the required libraries.

```bash
pip install -r requirements/requirements.txt
```

Run the complete project.

```bash
python run_all.py
```
