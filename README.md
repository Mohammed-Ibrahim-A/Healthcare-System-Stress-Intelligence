# Healthcare System Stress Intelligence

Healthcare System Stress Intelligence is a Machine Learning project that analyzes global mental health and healthcare data to predict healthcare stress levels. The project uses data preprocessing, visualization, regression, classification, and clustering techniques to generate meaningful insights that support healthcare decision-making.

## Features

- Data Preprocessing
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Data Visualization
- Regression Prediction
- Stress Classification
- Regional Clustering
- Feature Importance Analysis
- Business Insights
- Trend Analysis
- Report Generation

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib

## Folder Description

- **data/** – Contains the original dataset and processed datasets used for Machine Learning.
- **docs/** – Stores project documentation and supporting files.
- **images/** – Contains charts and visualizations generated during data analysis.
- **models/** – Stores trained Machine Learning models.
- **reports/** – Contains prediction results, business insights, feature importance, and trend analysis reports.
- **requirements/** – Contains the Python libraries required for the project.
- **src/** – Contains the complete source code for data preprocessing, visualization, model training, prediction, evaluation, clustering, and report generation.
